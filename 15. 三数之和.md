# 15. 三数之和
## 题目描述
## 题解
```python
# 方法一：暴力求解
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        l = len(nums)
        res =[]
        nums.sort()
        for i in range(l-2):
            for j in range(i+1,l-1):
                for k in range(j+1,l):
                    if nums[i] + nums[j] + nums[k] == 0 and [nums[i],nums[j],nums[k]] not in res:
                        res.append([nums[i],nums[j],nums[k]])
        return res


```
