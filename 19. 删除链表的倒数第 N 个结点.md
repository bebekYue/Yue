# 19. 删除链表的倒数第 N 个结点
## 题目描述

## 题解
```python
class Solution:
    def removeNthFromEnd(self, head: ListNode, n: int) -> ListNode:
        a = head
        b = head
    
        for i in range(n):
            if a.next:
                a = a.next
            else:
                return head.next
            
        while a.next:
            a = a.next
            b = b.next
        b.next = b.next.next
        return head
```
